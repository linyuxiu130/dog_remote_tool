## 功能概览

| 功能模块 | 当前能力 |
| --- | --- |
| 总览 | 显示当前设备信息、系统版本、小包版本、robot-launch 服务状态和基础连接状态。 |
| 录包 | 支持按主题配置 ROS 2 bag 录制，维护内置主题和设备差异化主题。 |
| 遥控 | 支持小狗/中狗远程遥控、键盘控制、robot_remote 协议检查和速度链路验证。 |
| 建图 | 支持开始建图、结束保存、取消建图、状态查询、历史地图预览、PCD/PGM/轨迹回传和建图临时 swap。 |
| 定位 | 支持单次定位、持续定位、关闭定位、定位数据回传，以及地图预览叠加定位轨迹。 |
| 导航 | 支持地图初始化、单点导航、多点导航、建图路线巡航、路网导航、循环模式、暂停/继续、停止、有图进桩和导航状态恢复。 |
| 路网编辑 | 支持加载、编辑、新建、保存、上传、导入、导出 `map.geojson`，支持节点/边校验、障碍膨胀显示和目标路线高亮。 |
| 文件管理 | 支持远端文件浏览、上传、下载和预览，便于调试日志、地图和配置文件。 |
| 远程访问 | 支持公网 SSH 映射、FRP 部署、远程脚本替换、NX 联网远程控制包部署和状态刷新。 |
| OTA / 线刷 | 支持小狗 L1 3588/NX 远程 OTA；支持中狗 3588 和中狗 NX full ZIP 远程 OTA；支持远程 OTA 目标的 deb/whl 文件、目录和压缩包小包部署；支持中狗和 L2 S100/NX 本机 USB 线刷预检、入口检测和刷写流程。 |
| 诊断 | 支持移动端/设备诊断信息采集、日志辅助分析和故障定位。 |

## 发版检查项

| 检查项 | 说明 |
| --- | --- |
| 更新版本号 | 修改 `VERSION` 和 `src/dog_remote_tool/__init__.py`，两处保持一致。 |
| 回归测试 | 运行 `PYTHONPATH=src pytest -q`，记录通过数量或失败原因。 |
| 完整打包 | 运行 `bash build/build_release.sh`，确认 `release/DogRemoteTool-v<version>-x86_64.run` 已生成。 |
| 产物校验 | 检查 `release/BUILD_MANIFEST.txt` 中的版本、构建时间、文件大小和 SHA256。 |
